# Healthcare Cost Analysis Using Medicare Inpatient Data 

## Overiew
Conducting SQL based analysis using BigQuery to evaluate 
medicare inpatient charges data and identify patterns in 
healthcare costs across providers and geographic regions.

## Objective
Support data driven decisions made by analysing large healthcare
datasets to uncover cost vaiation and utilization trends.

## Skills Demonstated
Data Aggregation and grouping
Filtering large datasets
Healthcare cost analysis
Analytical thinking
Translating data into insights

## Sample Insight
Preliminary analysis shows meaningful variation in average inpatient costs
across states. Suggesting opportunities for deeper evaluation of regional 
healthcare spending patterns.
